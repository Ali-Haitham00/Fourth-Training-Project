# Pricing Cards Project

This is a simple HTML & CSS project showcasing 3 pricing cards with modern design.  
Each card contains a label, price, features list, and a call-to-action button. The design highlights:

- Clean layout with centered cards
- Ribbon-style labels like "Basic" on top of cards
- Responsive design using Flexbox/Grid
- Hover effects on buttons for better UX

## Features

1. Three Pricing Cards
   - Each card has:
     - A label (e.g., Basic, Premium)
     - A price
     - A list of features
     - A sign-up button

2. Modern Design
   - Cards are centered on the page
   - Rounded corners and subtle shadows
   - Ribbon-style label with skewed background effect

3. Customizable
   - Colors, fonts, and card content can be easily modified in the CSS
   - Perfect for learning CSS positioning, pseudo-elements, and Flexbox/Grid layouts

## Usage

1. Open index.html in your browser
2. Edit the HTML & CSS to customize card content, colors, or layout
3. No dependencies required
